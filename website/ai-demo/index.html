API Setup

Add a free Anthropic key to unlock AI explanations on any question.

How to Get a Free API Key
  1. Open console.anthropic.com and create a free account.
  2. Go to Settings → API Keys.
  3. Click Create Key, name it (e.g. “SecPlus demo”), and copy the key.
  4. Paste it below — it starts with sk-ant-.
  5. Click Save Key, then return to Practice and tap Explain with AI.

Keys include free trial credits. This demo only sends your key to Anthropic — never to a backend of mine.
