# QuickQuote

Browser-based **invoice and quote generator** — build professional PDFs with no signup and no backend.

Try it live: **https://jacobbarrera2024-sketch.github.io/secplus-sensei/quick-quote/**

Portfolio project #3 — demonstrates fast delivery of a **small business web tool** (the most common entry-level freelance deliverable).

---

## What it demonstrates

- Vanilla JavaScript (no build step)
- Form-heavy business UX with live preview
- Line-item math, tax, totals
- `localStorage` draft persistence
- Print → Save as PDF export
- Responsive layout (editor + preview, mobile tabs)
- GitHub Pages deployment

---

## Features

- Invoice or quote mode
- Your business + client details
- Unlimited line items (description, qty, rate)
- Tax rate and notes / payment terms
- Three preview themes (Classic, Slate, Mint)
- Save draft locally · Load sample · Print / PDF

---

## Run locally

```bash
git clone https://github.com/jacobbarrera2024-sketch/secplus-sensei.git
cd secplus-sensei/quick-quote
npx --yes serve .
```

Open `http://localhost:3000`

---

## Deploy note

Edit files in `quick-quote/` only. The copy under `website/quick-quote/` is synced automatically when the portfolio site deploys.
