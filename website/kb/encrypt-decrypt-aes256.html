How to Encrypt/Decrypt Files with AES-256

Internal-style KB entry based on OpenSSL lab work and a Python/Tkinter encryption utility — written the way a help desk knowledge base should read.

Encrypt

openssl enc -aes-256-cbc -salt -pbkdf2 -in "file.zip" -out "file.zip.enc" -k "YourPassphrase"

Decrypt

openssl enc -d -aes-256-cbc -pbkdf2 -in "file.zip.enc" -out "file.zip" -k "YourPassphrase"

Common issues

SymptomCauseFix
bad decryptWrong password or -pbkdf2 mismatchMatch flags on both commands; verify passphrase
Corrupt outputFile transferred in text modeTransfer .enc as binary attachment
Unknown optionOlder OpenSSLUpgrade OpenSSL or drop -pbkdf2 on both sides

Security notes

  • Never send passphrase in the same channel as the encrypted file.
  • For routine use, prefer BitLocker or company-approved secure transfer over ad hoc OpenSSL.
