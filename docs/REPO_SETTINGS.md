# GitHub repo settings (2 minutes)

Do these once on github.com — the agent can't change them for you.

---

## Pin the repository

1. Your **GitHub profile** → **Customize your pins**
2. Pin **secplus-sensei**

---

## Repository About box

On the repo main page → click the **gear** next to "About":

| Field | Suggested value |
|-------|-----------------|
| **Description** | CompTIA Security+ desktop study app + developer portfolio. Electron, 500+ questions, exam sim. |
| **Website** | `https://jacobbarrera2024-sketch.github.io/secplus-sensei/` |
| **Topics** | `electron` `javascript` `portfolio` `desktop-app` `study-app` `security-plus` |

Topics help people discover the repo.

---

## Enable GitHub Pages

1. **Settings → Pages**
2. **Source** → **GitHub Actions**
3. **Actions** → **Deploy portfolio website** → **Run workflow**

Live URL: **https://jacobbarrera2024-sketch.github.io/secplus-sensei/**
